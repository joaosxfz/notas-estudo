# Resumo VisuAlg

Estrutura básica:

```bash
algoritmo "nome"

var
   // variáveis aqui

inicio
   // código aqui
fimalgoritmo
```

Variáveis:

```bash
var
  idade: inteiro
  altura: realidade 
  nome: caractere
  ativo: logico
```

Atribuição:

```bash
idade <- 18
nome <- "João"
```

Entrada e saída:

```bash
leia(nome)
escreva("Olá")
escreval("Olá") // pula linha
```

Condição (se):

```bash
se idade >= 18 entao
   escreva("maior")
senao
   escreva("menor")
fimse
```

Loop enquanto:

```bash
enquanto x < 10 faca
   x <- x + 1
fimenquanto
```

Loop para:

```bash
para i de 1 ate 10 faca
   escreval(i)
fimpara
```

Operadores:

```bash
+  -  *  /
>  <  >=  <=  =  <>
```

Dicas rápidas:
```bash
* leia = entrada
* escreva = saída
* <- = recebe valor
* se = decisão
* enquanto/para = repetição
```
